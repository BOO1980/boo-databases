Database Overview

  • A database is an organised collection of structured data stored electronically in a computer system
  • Computer requirements have increased to store larger volumes of data for fast retrieval
  • Before the database system was invented, the flat file structure was commonly used to store data
  • The flat file system consisted of fields and rows which are known as records
  • When the number of rows/records in the data files increased to millions the system can become unmanageable
  • In 1970s A Computer scientest invented the relational model for database management
  • The relational model deals with issues caused by the flat file model. The data is now organised in entities and attributes
